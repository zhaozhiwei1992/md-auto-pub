测试文章 - 微信公众号发布工具

这是使用 Python 脚本从 Markdown 发布到微信公众号草稿的测试文章。

功能介绍

本工具支持以下功能：

  • ✅ Markdown 自动转换为微信公众号格式
  • ✅ 自动提取标题、摘要
  • ✅ 图片自动上传
  • ✅ 自定义封面图
  • ✅ 直接保存为草稿

代码示例

def hello_wechat():
    print("Hello, WeChat!")

    # 创建 API 客户端
    api = WechatMPAPI(appid, secret)
    token = api.get_access_token()

    return token
